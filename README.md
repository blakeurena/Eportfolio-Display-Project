# Eportfolio Display Project

A responsive front-end project designed to showcase my developer portfolio across multiple device mockups. This project visually presents a portfolio website in **desktop, tablet, and mobile layouts**, demonstrating responsive design principles and modern front-end development practices.

The goal of this project is to highlight how a web application adapts to different screen sizes while maintaining a clean, professional user interface.

---

## Preview

This project displays a portfolio website within realistic device mockups including:

- Desktop
- Tablet
- Mobile

The layout helps visualize how the website behaves across devices, making it ideal for **portfolio presentation and project showcases**.

---

## Features

- Responsive multi-device display
- Desktop, tablet, and mobile mockups
- Clean UI presentation
- Optimized images for portfolio presentation
- Professional project display for developer portfolios
- JavaScript interactivity for enhanced user experience

---

## Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- Responsive design principles
- Image composition and UI mockups
